End-grain

Interactive exploration of cutting-board design operations.

Each operation has a focused harness — a constrained page that isolates one step of the pipeline so we can iterate on its math, rendering, and UI in isolation. The four harnesses below cover the full panel-producing pipeline. Read them in pipeline order.
