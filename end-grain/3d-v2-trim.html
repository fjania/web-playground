Testing: TrimPanel — crop to a clean rectangle

TrimPanel crops the upstream panel to an axis-aligned rectangle. It's first-class (appears in the timeline like any other feature, can run mid-pipeline, changes physical geometry that downstream features see). Three modes:

  • flush (default) — largest axis-aligned rectangle inscribed in the panel's top-face outline. Shaves triangular overhangs from a rip-angled glue-up while leaving the top fully covered.
  • rectangle — same rectangle but inscribed inside the INTERSECTION of top and bottom faces. At bevel ≠ 90° top and bottom footprints differ; this mode keeps trim cuts perpendicular to the panel surface at every y. Identical to flush at bevel = 90°.
  • bbox — user-specified bounds ?bounds=xMin,xMax,zMin,zMax. Omitted fields fall back to the panel's current bbox. Requests outside the panel clamp with status=warning.

Upstream: the harness runs a compose → cut → arrange triple with URL-tunable parameters (rip, bevel, slices, preset), then appends the TrimPanel. Example: ?rip=30&slices=6&preset=flipAlternate&mode=flush.

Expected in each tile:

  • Input — the upstream Arrange's reassembled panel, top-down. At rip > 0 this is a parallelogram with triangular overhangs; at bevel < 90° the outline is still a rectangle from above but the top and bottom faces differ.
  • Operation — the same panel with a dashed rectangle showing the imminent trim bounds and a dimmed overlay on the material about to be discarded (annular fill: panel bbox minus trim rect).
  • Output — the trimmed panel in 3D. Free orbit, home button, XYZ axis gizmo. The bbox matches appliedBounds exactly.

URL parameters

Trim parameters ?mode=flush (default) largest inscribed rect in the panel's top-face outline ?mode=rectangle intersect top + bottom footprints (matters at bevel != 90°) ?mode=bbox user-supplied bounds — omit fields to inherit the panel's extent on that axis ?bounds=-40,40,-150,150 xMin,xMax,zMin,zMax (mm) for bbox mode. Use _ for omitted fields: ?bounds=-40,40,_,_ trims X only. Upstream Cut (same as the Cut harness) ?rip=30 rip angle in degrees (default 0) ?bevel=60 bevel angle 45..90 (default 90) ?slices=6 how many slices to cut (default 6) Upstream Arrange (same as the Arrange harness) ?flip=1,3 rotate 180° on these slice indices ?spacer=1,3 insert spacers after these indices ?spacerWidth=10 width in mm (default 5) ?spacerSpecies=maple species (default walnut) ?preset=flipAlternate ?preset=spacerEveryRow:6 Multiple presets: ?preset=flipAlternate,spacerEveryRow:5 Examples Default flush trim on an angled glue-up: ?rip=30&slices=6 Trim to a specific size: ?mode=bbox&bounds=-40,40,-150,150 Bevel + rectangle mode: ?bevel=60&slices=6&mode=rectangle Arrange pattern + flush trim: ?rip=30&slices=8&preset=flipAlternate,spacerEveryRow:5
