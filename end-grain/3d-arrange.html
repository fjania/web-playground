Testing: Arrange — reassemble cut slices into a single panel

Arrange takes the slices produced by an upstream Cut, optionally applies per-slice PlaceEdits (rotate, shift, reorder) and SpacerInserts between slices, then stitches the result back into a single panel via cursor-slide concatenation. Presets (e.g. flipAlternate, rotate4way) are persistent features that expand into PlaceEdits / SpacerInserts at pipeline time.

All knobs drive via URL params — see the sidebar for the full reference. Example: ?slices=8&preset=flipAlternate for a 2×8 checker, ?flip=1,3&spacer=1&spacerWidth=10 for two flipped slices with a walnut spacer between slice 1 and slice 2.

Expected in each tile:

  • Input — the N slices flowing in from the upstream Cut, rendered top-down with gaps between each. Each slice is labelled with its index (0..N-1) so you can address it in ?flip= / ?shift= / ?spacer= params.
  • Operation — the reassembled panel with overlays showing what Arrange did: ↻ at each flipped slice's centroid (↻90°, ↻270° for non-flip rotations), ⇢ ±Δ for shifts, #N for reorders, and diagonal hatching on any inserted spacer volumes.
  • Output — the assembled panel in 3D. Free orbit, home button, XYZ axis gizmo in the upper-right. The panel's bbox reflects the final size including spacer additions.

URL parameters

Upstream Cut ?slices=4 how many slices to feed in (default 4) ?rip=30 rip angle in degrees (default 0) ?bevel=60 bevel angle 45..90 (default 90) Per-slice edits ?flip=1,3 rotate 180° on these slice indices ?shift=1:25 shift slice 1 by 25 mm along X (multiple: ?shift=1:25,3:-10) ?rotate=1:37 free-angle Y rotation (any degrees) e.g. ?rotate=0:15,2:-25 Caveat: mating breaks at non-90/180/270 angles — the rotated slice's cut faces no longer lie on the same plane family as its neighbours, so there'll be visible gaps. Useful for exploration, not for producing buildable panels. ?reorder=2:0 move slice at current position 2 to position 0. Chain with comma; each move is applied in order so later moves see the already-reordered sequence (like dragging slices one at a time). Example: ?reorder=2:0,3:1 Spacers (between slices) ?spacer=1 insert a 5 mm walnut spacer after slice 1 (multiple comma-separated) ?spacerWidth=10 width in mm ?spacerSpecies=maple species Presets (expand into edits/spacers at pipeline time) ?preset=flipAlternate ?preset=rotateAlternate:90 (90|180|270) ?preset=mirrorAlternate ?preset=rotate4way ?preset=shiftAlternate:25 (shift mm) ?preset=spacerEveryRow:6 (width mm) ?preset=spacerEveryRow:6:maple Multiple: ?preset=flipAlternate,spacerEveryRow:5 Example ?slices=4&flip=1,3&spacer=1 ?slices=8&preset=flipAlternate ?slices=8&preset=rotate4way
